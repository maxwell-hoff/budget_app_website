# How-it-works screen recordings

Drop `.mp4` files in **this folder** and they replace the animated mockups in the
"How it works" section automatically. No HTML edits needed.

- If a file exists here with the matching name, its video shows in place of the mockup.
- If a file is missing, the original animated mockup stays. So you can add recordings one at a time.
- The card keeps its rounded frame; whatever aspect ratio you record scales to fit the card width.

## Recording tips

- Record each clip at the **same aspect ratio** (16:10 recommended) so the section looks uniform.
- Any resolution works, but record at ~1600x1000 or larger so it stays sharp — the cards display up to ~900px wide.
- Keep clips short and loopable (5–15s). They autoplay muted and loop.
- Trim to `.mp4` (H.264 + AAC/none). That's what browsers play everywhere.
